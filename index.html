Firebase Console
  1. Aller sur console.firebase.google.com → projet coolcontrol
  2. Realtime Database → pour supprimer une salle : naviguer vers rooms/salle-X → clic droit → Supprimer
  3. La salle disparaît automatiquement du dashboard au prochain rafraîchissement
