Firebase Console — Étapes de sécurisation
  1. Aller sur console.firebase.google.com
  2. Projet coolcontrol → Authentication → Commencer → Email/Password → Activer
  3. Ajouter un utilisateur : admin@coolcontrol.local
  4. Realtime Database → Règles → Coller les règles ci-dessous
  5. Publier
